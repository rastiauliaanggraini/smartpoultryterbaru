# Project Blueprint

## Overview

This project is a Flutter application that helps predict egg production in a poultry farm based on various environmental factors.

## Features

### Implemented

*   **Dashboard Page:** A user interface for inputting manual data and viewing sensor readings.
*   **Input Fields:**
    *   Amount of chicken
    *   Amount of Feeding
    *   Ammonia
    *   Light Intensity
    *   Noise
*   **Sensor-like display:**
    *   Temperature
    *   Humidity
*   **Prediction:**
    *   A button to trigger the prediction.
    *   Display for the total predicted egg production.

### Current Plan

*   Create the initial dashboard UI.
*   Integrate the dashboard into the main application.
